# Retinal OCT Disease Classification – 8 Classes (C8 Dataset)

This project builds a deep learning classifier using OCT (Optical Coherence Tomography) images 
to detect 8 different retinal conditions:

- AMD
- CNV
- CSR
- DME
- DR
- DRUSEN
- MH
- NORMAL

Model: MobileNetV2 / EfficientNetB0  
GUI: Streamlit  
Dataset Size: 24,000 images  
Split: Train · Val · Test

Project includes:
- Data preprocessing
- Transfer learning
- Model training
- Streamlit GUI
- Demo screenshots
